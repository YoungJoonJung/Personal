ETF 주가 현황

• 전체보기 선택 시: 표만 표시(개별 그래프/카드 없음), 구분 컬럼 포함(동일 구분 셀 병합), 구분별 총수익 합계 + 전체 총수익 합계 표시
• 전체보기 표에서 구분(DC/IRP/연금저축)을 클릭하면, 콤보박스가 해당 구분으로 바뀌고 화면이 전환됩니다.
• DC/IRP/연금저축 선택 시: 종목별 카드(차트/현재가/매수가/주식수/요약) + 상단 요약표 표시, 요약표에서 종목명 클릭 시 해당 종목 그래프(카드)로 자동 스크롤
• 현재가/증감은 Google Apps Script(JSONP)로 조회되며, 요약표에서는 현재가/증감 컬럼으로 분리 표시
• 매수가는 입력 시 자동 콤마 적용(최대 100,000), 주식수는 최대 9,999
